# フリック入力インターフェース

物理ボタン + 静電容量タッチパッドで日本語フリック入力を実現する USB HID キーボードデバイスです。  
M5Stack **ATOM S3** (ESP32-S3) を使用し、PC に接続するだけで日本語 IME へローマ字を送信できます。

## 概要

スマートフォンのフリック入力と同じ操作感で日本語を入力できる **物理キーボードデバイス** です。

- タッチパッドに指を置いて行を選択し、方向ボタンでフリック方向を指定します。
- ATOM S3 の内蔵ボタンで **PAGE1（通常）** ↔ **PAGE2（濁音・半濁音・記号）** を切り替えられます。
- USB HID キーボードとして動作するため、ドライバのインストールは不要です。

## ハードウェア構成

| コンポーネント | 型番 / スペック | 役割 |
|---|---|---|
| マイコン | M5Stack ATOM S3 (ESP32-S3) | メイン制御・USB HID 送信・LCD 表示 |
| タッチセンサ #1 | Adafruit MPR121 (I2C: 0x5A) | 電極 0–3 → パッド 0–3（4行分） |
| タッチセンサ #2 | Adafruit MPR121 (I2C: 0x5B) | 電極 0–4 → パッド 4–8（5行分） |
| GPIO エクスパンダ | MCP23017 (I2C: 0x20) | 方向ボタン・BS・スペースボタン入力 |
| 方向スイッチ | タクタイルスイッチ × 4 | ↑ ← ↓ → フリック方向入力 |
| BS ボタン | タクタイルスイッチ × 1 | バックスペース |
| スペースボタン | タクタイルスイッチ × 1 | スペース / 変換候補送り |

## 配線図

### I2C バス共通配線

```
ATOM S3  SDA → G2
ATOM S3  SCL → G1
ATOM S3  3V3 → 各デバイス VCC
ATOM S3  GND → 各デバイス GND
```

### I2C アドレス設定

| デバイス | ADDR ピン設定 | I2C アドレス |
|---|---|---|
| MPR121 #1 | ADDR → GND | 0x5A |
| MPR121 #2 | ADDR → VCC (3V3) | 0x5B |
| MCP23017 | A0, A1, A2 → GND | 0x20 |

### MCP23017 GPA ピンアサイン（active-LOW）

| GPA ピン | 接続先 | 機能 |
|---|---|---|
| GPA0 | ↑ スイッチ（反対端 GND） | フリック上 |
| GPA1 | ← スイッチ（反対端 GND） | フリック左 |
| GPA2 | ↓ スイッチ（反対端 GND） | フリック下 |
| GPA3 | → スイッチ（反対端 GND） | フリック右 |
| GPA4 | BS ボタン（反対端 GND） | バックスペース |
| GPA5 | SP ボタン（反対端 GND） | スペース |

## ビルド・書き込み手順

### 前提条件

- [Visual Studio Code](https://code.visualstudio.com/) + [PlatformIO IDE 拡張機能](https://platformio.org/install/ide?install=vscode) がインストール済みであること
- または PlatformIO Core (CLI) がインストール済みであること

### VS Code + PlatformIO IDE の場合

1. このリポジトリをクローンして VS Code で開く
2. PlatformIO のサイドバーから **Build** → **Upload** を実行

### CLI の場合

```bash
# リポジトリのクローン
git clone https://github.com/furukawa1020/flicki-pythsical-interface.git
cd flicki-pythsical-interface

# ビルド
pio run

# ATOM S3 に書き込み
pio run --target upload

# シリアルモニタ (デバッグ時)
pio device monitor
```

## 使い方

### 基本操作

1. **行の選択**: タッチパッドに指を置くと LCD に行が表示されます。
2. **文字の確定**:
   - 方向ボタン（↑← ↓→）を押すとフリック方向に対応した文字を送信します。
   - パッドを離すと中央（1文字目）を送信します。
   - パッドを押したまま 250ms 経過すると中央（1文字目）を自動送信します。
3. **バックスペース**: BS ボタンを押します（長押しで連続削除）。
4. **スペース**: SP ボタンを押します（変換候補の送りにも使用可）。
5. **ページ切替**: ATOM S3 の内蔵ボタン（BtnA）を押すと PAGE1 ↔ PAGE2 が切り替わります。  
   LCD の右上に **P1**（シアン）/ **P2**（マゼンタ）でページが表示されます。

### フリック方向と文字の関係

```
      [ 上 ]
[ 左 ][ 中 ][ 右 ]
      [ 下 ]
```

例:「か行」パッドの場合  
→ 中央=**か** / 上=**き** / 左=**く** / 下=**け** / 右=**こ**

|---|---|---|---|---|---|
| あ行 (0) | あ | い | う | え | お |
| か行 (1) | か | き | く | け | こ |
| さ行 (2) | さ | し | す | せ | そ |
| た行 (3) | た | ち | つ | て | と |
| な行 (4) | な | に | ぬ | ね | の |
| は行 (5) | は | ひ | ふ | へ | ほ |
| ま行 (6) | ま | み | む | め | も |
| や行 (7) | や | い | ゆ | え | よ |
| ら行 (8) | ら | り | る | れ | ろ |

### PAGE 2（濁音・半濁音・記号 — マゼンタ表示）

| パッド | 中央 | 上 | 左 | 下 | 右 |
|---|---|---|---|---|---|
| わ行 (0) | わ | ゐ | (ゑ相当) | を | ｳﾞ相当 |
| が行 (1) | が | ぎ | ぐ | げ | ご |
| ざ行 (2) | ざ | じ | ず | ぜ | ぞ |
| だ行 (3) | だ | ぢ | づ | で | ど |
| ば行 (4) | ば | び | ぶ | べ | ぼ |
| ぱ行 (5) | ぱ | ぴ | ぷ | ぺ | ぽ |
| ん (6) | ん | ん | ん | ん | ん |
| 小文字 (7) | っ | ぁ | ぅ | ぉ | ぃ |
